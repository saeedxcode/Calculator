# 🧮 Calculator Pro

A feature-rich console-based calculator application built with C# and .NET 8. Perform basic and advanced mathematical operations with a beautiful user interface.

## ✨ Features

### Basic Operations
- ➕ **Addition** - Add two numbers
- ➖ **Subtraction** - Subtract two numbers
- ✖️ **Multiplication** - Multiply two numbers
- ➗ **Division** - Divide two numbers (with zero division handling)

### Advanced Operations
- 🔢 **Power** - Calculate x^y
- √ **Square Root** - Calculate √x (with negative number handling)
- 💯 **Percentage** - Calculate percentage of a number

### Additional Features
- 📜 **History** - View all previous calculations
- 🎨 **Colorful Console UI** - Enhanced visual experience
- 🛡️ **Input Validation** - Prevents crashes from invalid inputs
- 🔄 **Continuous Operation** - Perform multiple calculations without restarting
- ❌ **Graceful Exit** - Clean exit option
